
# Hospital Management System (Java)

A simple object-oriented Hospital Management System built in Java.  
This project demonstrates the use of **inheritance, abstract classes, polymorphism**, and basic program structure.

---

## 🚀 Features

- Base abstract class `Human` with common attributes  
- Subclasses such as `Employee`, `Doctor`, `Patient`, etc.  
- Overridden `showInfo()` methods for displaying details  
- Clean class hierarchy for easy extension  
- Example `main()` program that runs the system
